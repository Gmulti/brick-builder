## :fr: Documentation

Ce module contient toute la logique métier de Drag'n'Drop pour un builder construit sur l'architecture : section - colonne - composant

L'index permet d'exposer la totalité de la lib.

Les dépendances :

-   `redux`
-   `lodash`
